# 独自ドメイン設定手順（flyer.riceparrow.com → Vercel）

チラトラ（Vercel上のNext.jsアプリ）を、独自ドメインのサブドメイン
`flyer.riceparrow.com` で見られるようにする手順。

## 前提となる考え方
- `riceparrow.com`（apex）は**レンタルサーバ**（LP）に向いている。**これは触らない**。
- チラトラは**Vercelに置いたまま**、サブドメイン `flyer.` だけをVercelに向ける。
- 仕組み: DNSに「`flyer` は Vercel を見よ」というCNAMEを1本足す → VercelがSSL証明書を自動発行。

```
riceparrow.com            → レンタルサーバ（A レコード。既存のまま）
flyer.riceparrow.com      → Vercel（CNAME を新規追加）  ← 今回やるのはココ
```

---

## 手順

### 1. Vercel側：ドメインを登録
1. Vercel → チラトラのプロジェクト → **Settings → Domains**
2. `flyer.riceparrow.com` を入力して **Add**
3. Vercelが「追加すべきDNSレコード」を表示する。通常：
   - Type: **CNAME** / Name: `flyer` / Value: **`cname.vercel-dns.com`**
   - （この値は画面の指示を正とする。将来変わる可能性があるため）

### 2. お名前.com側：DNSレコードを追加
1. お名前.com Navi → **ドメイン → DNS → DNSレコード設定**（対象: `riceparrow.com`）
2. レコードを1本追加：
   | 項目 | 値 |
   |---|---|
   | ホスト名 | `flyer` |
   | Type | `CNAME` |
   | VALUE | `cname.vercel-dns.com`（手順1でVercelが示した値） |
   | TTL | 既定のまま |
3. 保存。**apexの A レコード（レンタルサーバ向け）は消さない・変えない**。

### 3. 反映を待つ（数分〜最大数十分）
- DNSが伝播すると、Vercelの Domains 画面が **Valid / 緑チェック**になり、**SSL証明書（Let's Encrypt）が自動発行**される。
- 確認: ブラウザで `https://flyer.riceparrow.com` を開く。鍵マーク（https）が付けばOK。

### 4. 認証まわりの設定を新URLに合わせる（重要）
サブドメインで開くと、ログイン後のリダイレクト先が変わるので合わせる。
1. **Vercel → Settings → Environment Variables**
   - `NEXT_PUBLIC_SITE_URL` を `https://flyer.riceparrow.com` に更新 → **再デプロイ**
2. **Supabase → Authentication → URL Configuration**
   - **Site URL** を `https://flyer.riceparrow.com` に
   - **Redirect URLs** に `https://flyer.riceparrow.com/auth/confirm` を追加
     （既存の Vercel の `*.vercel.app` URLは残しておいてよい）

### 5. 動作確認
- `https://flyer.riceparrow.com` にアクセス → ログイン画面が出る
- ログイン → トップ（地図）に遷移する
- パスワードログイン・地図・一覧が動く

---

## つまずきポイント
- **`This domain is not configured` が消えない**: DNS伝播待ち。数分〜数十分。`nslookup flyer.riceparrow.com` でCNAMEが返るか確認。
- **apexサイト（LP）が落ちた**: apexの A レコードを触ってしまった可能性。レンタルサーバのIPに戻す。
- **ログイン後にエラー/ループ**: 手順4の `NEXT_PUBLIC_SITE_URL` 更新＆再デプロイ、SupabaseのRedirect URL登録を確認。
- **サブドメイン名を変えたい**: `flyer` の部分は任意（例 `app`, `chiratra`）。手順1・2のホスト名を合わせるだけ。
